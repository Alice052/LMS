simple prototye for a library made using java

##features
addition of records
deletion of records
display of records

books -> has all the getter and setter 
management -> has all the logic for addition, deletion, displaying
main -> entry point of the program

to compile: javac -d out books/*.java management/*.java main/*.java
            java -cp out main.Main

*IMPORTANT*
1.all the changes are visible in the system_log.txt file

2.when restarting the program; delete all the old records from system_log.txt file
